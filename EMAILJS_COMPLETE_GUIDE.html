Add Email Service (Gmail)

After logging in to EmailJS dashboard:

  • Click "Email Services" in the left menu
  • Click "Add New Service" button
  • Select "Gmail" from the list
  • Click "Connect Account"
  • Log in with your Gmail: princejheckjuan023@gmail.com
  • Allow EmailJS permissions
  • Click "Create Service"
✅ Copy your Service ID! It looks like: service_abc1234
3

Create Email Template

Now let's create the email template:

  • Click "Email Templates" in the left menu
  • Click "Create New Template"
  • Give it a name: Contact Form

In the template editor, use this content:

Subject:

New Message from {{from_name}} - {{subject}}

Content (Body):

Hello Prince, You have a new contact form submission: Name: {{from_name}} Email: {{from_email}} Subject: {{subject}} Message: {{message}} --- Sent from your Portfolio Website
  • Click "Save"
✅ Copy your Template ID! It looks like: template_xyz5678
4

Get Your Public Key

  • Click "Account" in the left menu (or your profile icon)
  • Look for "Public Key" section
  • You'll see something like: AbC123XyZ456
✅ Copy your Public Key! Keep all three keys ready!
5

Configure Your Portfolio

Now we'll add your keys to the website:

1. Open your project folder:

C:\Users\prince\OneDrive\Dokumen\portfolio-website-design\

2. Find and open this file in Notepad:

emailjs-config.js

3. Replace the placeholders with YOUR keys:

// EmailJS Configuration const EMAILJS_CONFIG = { publicKey: 'YOUR_PUBLIC_KEY_HERE', // Replace with your Public Key serviceId: 'YOUR_SERVICE_ID_HERE', // Replace with your Service ID templateId: 'YOUR_TEMPLATE_ID_HERE' // Replace with your Template ID };

Example (with fake keys):

// EmailJS Configuration const EMAILJS_CONFIG = { publicKey: 'AbC123XyZ456', serviceId: 'service_abc1234', templateId: 'template_xyz5678' };

4. Save the file (Ctrl + S)

6

Activate EmailJS on Your Site

Double-click this file in your project folder:

activate-emailjs.bat

This will automatically:

  • ✅ Update contact.php with EmailJS integration
  • ✅ Update script.js with email sending code
  • ✅ Back up your old files
✅ Done! You'll see a success message when complete.

Troubleshooting

No email received?

  • ✅ Check your Gmail SPAM folder
  • ✅ Verify all 3 keys are correctly entered in emailjs-config.js
  • ✅ Make sure you clicked "Save" on the template
  • ✅ Check browser console (F12) for errors
  • ✅ Make sure message is at least 10 characters

Still having issues?

  • Open browser console (press F12)
  • Look for red error messages
  • Take a screenshot and let me know!
