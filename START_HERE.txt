📞 COMMON QUESTIONS:

Q: How do I run it locally?
A: php -S localhost:8000 then open http://localhost:8000

Q: How do I customize my name?
A: Edit index.php line ~17, change "John Doe" to your name

Q: How do I add projects?
A: Copy a project card in index.php and modify the content

Q: How do I change colors?
A: Edit styles.css, modify --primary, --secondary, --accent variables

Q: How do I deploy it?
A: Read DEPLOYMENT.md for step-by-step instructions

Q: What if forms don't work?
A: Ensure PHP is running and directory is writable (see TESTING.md)

Q: Can I use it on shared hosting?
A: Yes, any hosting with PHP 7.0+ support works (see DEPLOYMENT.md)
