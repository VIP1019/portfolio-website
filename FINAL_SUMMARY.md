## Navigation System - How It Works

### Header Menu (Appears on Every Page)
```
[Your Logo] Prince Jheck | Home | About me | Skills | Projects | Contact | [Social Icons]
```

When users click a button in the header:
- **Home** → Takes them to `index.php` (main page)
- **About me** → Takes them to `about.php` (about page)
- **Skills** → Takes them to `skills.php` (skills page)
- **Projects** → Takes them to `projects.php` (projects page)
- **Contact** → Scrolls to contact form on home page
- **Social Icons** → Opens your social media in new tabs

All pages are styled identically with your professional design.

## Support & Troubleshooting

### Common Issues

**Links not working?**
- Make sure all `.php` files are in the same directory
- Check that filenames match exactly in links

**Styles not showing?**
- Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
- Make sure `styles.css` is in same directory as `.php` files

**Forms not submitting?**
- Ensure PHP is enabled on your server
- Check that `contact-handler.php` is in the same directory

**Mobile looks weird?**
- This is likely a browser zoom issue
- Try resetting zoom to 100%
- Check in actual mobile device
