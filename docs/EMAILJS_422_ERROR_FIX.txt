  EMAILJS 422 ERROR - COMMON CAUSES & FIXES
═══════════════════════════════════════════════════════════════

A 422 error means EmailJS rejected the request. Here's what to check:

═══════════════════════════════════════════════════════════════
  MOST COMMON ISSUE: Template "To Email" Setting
═══════════════════════════════════════════════════════════════

In your EmailJS template, check the "To Email" field on the right side:

❌ WRONG: {{email}}  (trying to use a variable)
✅ RIGHT: princejheckjuan023@gmail.com  (your actual email)

OR if you want dynamic:
✅ RIGHT: Use {{email}} but make sure to set a DEFAULT email in template settings


═══════════════════════════════════════════════════════════════
  QUICK FIX STEPS:
═══════════════════════════════════════════════════════════════

1. Go to: https://dashboard.emailjs.com/admin/templates/template_14hkl4h

2. Look at the right sidebar under "Settings"

3. Find "To Email" field

4. Change it to: princejheckjuan023@gmail.com

5. Click SAVE

6. Try submitting testimonial again


═══════════════════════════════════════════════════════════════
  OTHER POSSIBLE CAUSES:
═══════════════════════════════════════════════════════════════

✓ Template expects variables that aren't being sent

2. Look for the console.log message showing the params being sent
3. Compare with what your template expects
4. Check EmailJS dashboard for error details
