📧 Testimonial Email Setup

Send emails automatically when visitors leave testimonials!

✅ What's Already Done:

  • EmailJS SDK added to your website
  • EmailJS configuration file is ready
  • Testimonial form integrated with EmailJS
  • Code updated to send emails on testimonial submission

⚠️ What You Need to Do:

Create a new email template in your EmailJS account specifically for testimonials.

1 Login to EmailJS

Go to your EmailJS dashboard:

Open EmailJS Dashboard

Use the same account you used for your contact form.

2 Create New Email Template

Follow these steps:

  • Click on Email Templates in the left sidebar
  • Click Create New Template button
  • You'll see the template editor

3 Set Template Name and ID

Important: The Template ID must be exactly:

template_testimonial

You can name it whatever you like, but the ID must match the code.

4 Configure the Template

📝 Template Subject Line:

New Testimonial from {{from_name}} - {{stars}}

📄 Template Body (HTML or Plain Text):

Hello, You have received a new testimonial on your portfolio website! ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 👤 From: {{from_name}} 📧 Email: {{from_email}} ⭐ Rating: {{stars}} ({{rating}}/5) 🕒 Submitted: {{timestamp}} ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 💬 Feedback: {{feedback}} ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ This testimonial has been automatically saved to your website. Best regards, Your Portfolio Website

Available Variables:

  • {{from_name}} - Visitor's name
  • {{from_email}} - Visitor's email address
  • {{rating}} - Numeric rating (1-5)
  • {{stars}} - Star emoji representation (⭐⭐⭐⭐⭐)
  • {{feedback}} - The testimonial message
  • {{timestamp}} - Date and time of submission

5 Set Email Recipients

In the To email field, enter the email address where you want to receive testimonial notifications.

This is usually your personal or business email address.

6 Save the Template

Click the Save button to save your template.

Make sure the Template ID is template_testimonial

7 Test It Out!

Now you're ready to test:

  1. Go to your portfolio website
  2. Navigate to the "Feedback & Testimonials" section
  3. Fill out the testimonial form
  4. Submit it
  5. Check your email inbox for the notification!

🎉 What Happens When Someone Submits a Testimonial:

  1. An email is sent to you via EmailJS with all the details
  2. The testimonial is saved to your website's JSON file
  3. It appears on your homepage immediately
  4. The visitor sees a success message

🔧 Troubleshooting:

  • Emails not sending? Check that Template ID is exactly template_testimonial
  • Check browser console: Press F12 and look for error messages
  • Verify EmailJS quota: Free plan has 200 emails/month limit
  • Check spam folder: Sometimes emails go to spam
