📋 How to Paste HTML in EmailJS

Step-by-step visual guide

1 Choose Your Template Style

📝 Simple Text

  • Plain text format
  • Works everywhere
  • Quick to setup
  • Basic formatting

Good for: Quick setup

🎨 Beautiful HTML

  • Professional design
  • Colors & gradients
  • Styled layout
  • Interactive buttons

Good for: Professional look

2 Open EmailJS Template Editor

  1. Go to EmailJS Dashboard
  2. Click Email Templates in sidebar
  3. Click Create New Template

You'll see a screen like the screenshot you showed me!

The editor has two tabs at the top: Desktop and Mobile

3 Set Template ID

IMPORTANT: At the top of the page, set the Template ID to:

template_testimonial

This MUST match exactly what's in your code!

4 Set Subject Line

In the Subject field at the top, paste this:

🎉 New Testimonial from {{from_name}} - {{stars}}

5 Add the HTML Content

⚠️ For HTML Template:

  1. Open the file: TESTIMONIAL_EMAIL_TEMPLATE_HTML.html
  2. Select ALL the code (Ctrl+A)
  3. Copy it (Ctrl+C)
  4. Go back to EmailJS editor
  5. Click in the large content area (where you saw the HTML code in your screenshot)
  6. Delete everything that's there (Ctrl+A, then Delete)
  7. Paste your HTML (Ctrl+V)

✅ For Simple Text Template:

  1. Open the file: TESTIMONIAL_EMAIL_TEMPLATE_SIMPLE.txt
  2. Copy the "OPTION A: SIMPLE TEXT VERSION" content
  3. Paste it in the EmailJS content area

6 The Editor Should Look Like This

Based on your screenshot, you'll see:

  • Line numbers on the left (1, 2, 3, etc.)
  • Your HTML code with syntax highlighting (colors)
  • Variables like {{name}}, {{message}} will be highlighted
  • A preview pane on the right showing Desktop/Mobile view

💡 Pro Tip:

The preview on the right will show how your email looks!

Variables like {{from_name}} will show as placeholders in the preview.

7 Set the Recipient

Scroll down and find the To email field.

Enter YOUR email address where you want to receive testimonial notifications.

your-email@example.com

8 Save the Template

  1. Review your template content
  2. Make sure Template ID is template_testimonial
  3. Click the Save button (usually at the bottom or top right)

🎉 That's it! You're done!

📋 Available Variables Reference

These variables will be automatically replaced with real data:

Variable What it shows Example
{{from_name}} Person's name John Doe
{{from_email}} Person's email john@example.com
{{rating}} Numeric rating 5
{{stars}} Star emojis ⭐⭐⭐⭐⭐
{{feedback}} Testimonial message Great work!
{{timestamp}} Date & time 2/23/2026, 8:40:18 AM

9 Test It!

  1. Go to your portfolio: http://localhost:3000/index.php
  2. Scroll to "Feedback & Testimonials" section
  3. Fill out the form with test data
  4. Click "Submit Feedback"
  5. Check your email inbox! 📧

🎊 You're All Set!

Your testimonial form will now send beautiful email notifications!
