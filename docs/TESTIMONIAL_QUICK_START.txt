🚀 QUICK SETUP (Follow these steps):

STEP 1: Open EmailJS Dashboard
    → Go to: https://dashboard.emailjs.com/admin
    → Login with your account

STEP 2: Create New Template
    → Click "Email Templates" (left sidebar)
    → Click "Create New Template" button

STEP 3: Set Template ID
    ⚠️ IMPORTANT: Set Template ID to exactly this:
    
    template_testimonial
    
    (Must match exactly - no spaces, no typos!)

STEP 4: Configure Template

    Subject Line:
    ─────────────────────────────────────────────────────────
    New Testimonial from {{from_name}} - {{stars}}
    ─────────────────────────────────────────────────────────

    Body/Content:
    ─────────────────────────────────────────────────────────
    Hello,

    You have received a new testimonial on your portfolio website!

    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    👤 From: {{from_name}}
    📧 Email: {{from_email}}
    ⭐ Rating: {{stars}} ({{rating}}/5)
    🕒 Submitted: {{timestamp}}

    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    💬 Feedback:
    {{feedback}}

    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    This testimonial has been automatically saved to your website.

    Best regards,
    Your Portfolio Website
    ─────────────────────────────────────────────────────────

STEP 5: Set Recipient Email
    → In "To email" field, enter YOUR email address
    → This is where you'll receive testimonial notifications

STEP 6: Save Template
    → Click "Save" button
    → Done! 🎉

🎯 WHAT HAPPENS WHEN SOMEONE SUBMITS A TESTIMONIAL:

    1. User fills out the form
       ↓
    2. EmailJS sends you an email notification 📧

       ↓
    4. Testimonial appears on your website
       ↓
    5. User sees success message

❓ TROUBLESHOOTING:

    Problem: Emails not sending
    Solution: 
    - Check Template ID is EXACTLY "template_testimonial"
    - Press F12 in browser to see console errors
    - Verify EmailJS service is active

    Problem: Form works but no email
    Solution:
    - Make sure you created the template in EmailJS
    - Check spam/junk folder
    - Verify "To email" is set in template
