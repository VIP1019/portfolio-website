## Common Questions

### Q: Can I add more projects?
**A:** Yes! Open `projects.php` and duplicate a project card. Update the text and images.

### Q: How do I add my photo?
**A:** Open `about.php` and look for the image section. Add your photo path.

### Q: Can I change the design?
**A:** Yes! Edit `styles.css` to customize colors, fonts, spacing, etc.

### Q: How do people contact me?
**A:** They fill the contact form on the home page. Data is processed by `contact-handler.php`.

### Q: What if forms don't work?
**A:** Make sure:
1. You're using PHP (not just HTML)
2. All `.php` files are in the same folder
3. Your hosting has PHP enabled

## Support

### If Something Breaks

1. **Check file paths** - Make sure all files are in the same folder
2. **Clear cache** - Hard refresh: Ctrl+F5 or Cmd+Shift+R
3. **Check console** - Open developer tools (F12) for error messages
4. **Verify files exist** - Make sure all `.php` files exist

### Troubleshooting

| Issue | Solution |
|-------|----------|
| Links don't work | Check that `.php` files exist and names match |
| Styles don't show | Clear cache and refresh |
| Forms don't send | Verify PHP is enabled on your server |
| Mobile looks weird | Check responsive design in different screen sizes |
