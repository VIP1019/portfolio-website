STEP 1: Open EmailJS
   → https://dashboard.emailjs.com/admin/templates/new

STEP 2: Set Template ID
   → Template ID: template_testimonial
   
STEP 3: Set Subject
   → Subject: 🎉 New Testimonial from {{from_name}} - {{stars}}

STEP 4: Choose Your Style

   OPTION A - Beautiful HTML (Recommended):
   ─────────────────────────────────────────
   1. Open: TESTIMONIAL_EMAIL_TEMPLATE_HTML.html
   2. Copy ALL the code (Ctrl+A, Ctrl+C)
   3. Paste in EmailJS content editor (Ctrl+V)
   
   OPTION B - Simple Text:
   ─────────────────────────────────────────
   1. Open: TESTIMONIAL_EMAIL_TEMPLATE_SIMPLE.txt
   2. Copy the text version
   3. Paste in EmailJS content editor

STEP 5: Set Recipient
   → To email: YOUR_EMAIL@example.com

STEP 6: Save
   → Click "Save" button

STEP 7: Test!
   → Go to your website
   → Submit a testimonial
   → Check your email!
